# Ticket Application

This is a mini bus application. You can get on a bus which is specified on the page. You can check your account informations and create accounts.

How to execut it :

 `mvn clean package`

`docker-compose up`


For this application following dependencies used :

- Docker
- Java 1.17
- Spring Boot
- Spring Data JPA
- Restful API
- Thymeleaf
- MySQL
